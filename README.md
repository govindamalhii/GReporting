GReporting** is a professional web-application designed to help cybersecurity researchers and beginners create high-quality, structured vulnerability reports. This project was developed as a final year project to bridge the gap between finding a bug and reporting it effectively.

---

üöÄ Features
* **Live Preview:** See your report update in real-time as you type.
* **Auto-Severity Suggestion:** Automatically suggests severity (Critical/High/Med) based on the vulnerability type.
* **Professional Templates:** Pre-defined descriptions for SQLi, XSS, IDOR, and more.
* **One-Click Export:** Download reports as PDF or copy them in Markdown format for HackerOne/Bugcrowd.
* **Modern UI:** Sleek dark-mode interface built with Tailwind CSS.
